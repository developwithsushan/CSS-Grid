CSS Grid:
Css Grid Layout is a brand new module that brings a two-dimensional grid system to CSS for the first time.
CSS Grid replaces float layouts, using less and more readable and logical css and html
CSS grid work perfectly together with flexbox, which is best to handle one-dimensional components and layouts
CSS grid completely changes the way that we envision and build two dimensional layout.
